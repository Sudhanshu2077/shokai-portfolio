# shokai-portfolio

This is a Vite + React + Tailwind portfolio project for RafeFx Studio (Shokai / Sudhanshu Gayake).

## Quick start

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
```

This project is ready to deploy on Vercel. Connect your GitHub repository to Vercel and deploy the `main` branch.
